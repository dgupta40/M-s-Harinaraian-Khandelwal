# Project Guidelines

## Skills

Before writing any frontend, React, or web UI code, read and follow the guidelines in these skill files:

- `.agents/skills/frontend-design/`
- `.agents/skills/vercel-react-best-practices/`
- `.agents/skills/web-design-guidelines/`

## Workflow

1. Before making any changes, audit the existing codebase against the skill guidelines above
2. Produce a review summary listing what violates or deviates from the best practices
3. Propose a rebuild plan with prioritized changes
4. Wait for approval before implementing changes
